# leedshack2026-prototype

## Prerequisites

Ensure you have the following installed on your machine:

- Node.js & pnpm

- Python

- Uvicorn (for the Python API)

## Starting the app

run in root directory to start app

```pnpm install```

```pnpm run dev```

run in model folder to start weather prediction api

```uvicorn main:app --reload```


Enjoy on http://localhost:3000/
